# Nu Frequencies

Application web de respiration consciente et de fréquences binaurales.

👉 **[Lancer l'application](https://nufrequencies.xyz/)**

## Fonctionnalités
- Modes binauraux (Relaxation, Concentration, Clarté, Sommeil)
- Fréquences pures (432 Hz, 440 Hz)
- Guide visuel de respiration (cohérence cardiaque, respiration carrée, 4-7-8)
- Sons générés en temps réel, sans téléchargement
- Design Glassmorphism
- Bilingue (FR / EN)

## Déploiement
PWA d'un seul fichier (`index.html`), publiée via **GitHub Pages** sur le domaine
personnalisé **nufrequencies.xyz** (`CNAME` + `.nojekyll`).

---
*Une application iOS native distincte (StoreKit) est développée séparément ; ce dépôt
correspond à la version web.*
